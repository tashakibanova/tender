# Tender Procurement Bot

Десктопная система на Python для автоматизации участия в тендерах с обязательным human-in-the-loop.
Все данные хранятся локально, а каждое критическое действие требует подтверждения пользователя.

## Возможности
- Мониторинг площадок через UI браузера (Selenium/Playwright через ручную автоматизацию).
- Поиск по ключевым словам из профиля.
- Сохранение найденных лотов в локальный JSON.
- Desktop-уведомления о новых лотах.
- Анализ технических заданий (PDF/DOCX/TXT) с локальным AI (spaCy).
- Подготовка документов по шаблонам DOCX с переменными `{{НАИМЕНОВАНИЕ}}`, `{{ИНН}}`, `{{ЦЕНА}}`.
- Экспорт отчетов в Excel и JSON.

## Запуск
```bash
cd procurement_bot
python main.py
```

### Установка модели spaCy
```bash
python -m spacy download ru_core_news_sm
```

### Установка браузеров Playwright
```bash
python -m playwright install
```

### Selenium
Для Selenium требуется установленный драйвер браузера (например, ChromeDriver) в PATH.

## Локальные данные
- `config/profile.json` — профиль пользователя.
- `data/lots/` — найденные лоты.
- `data/templates/` — шаблоны документов.
- `data/reports/` — локальные отчеты.

## Принципы
- Нет автоматической подачи заявок, подписей и платежей.
- Только десктопная версия и локальное хранение.
- Все действия на русском языке.
